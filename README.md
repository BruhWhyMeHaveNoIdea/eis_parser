# EIS Parser Site

Сайт для запуска парсера данных о закупках.

## Требования

- Python 3.10+
- pip

## Установка

1. Клонируйте репозиторий (если ещё не сделано)

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
```bash
# Windows
.\venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

### Запуск сайта

Перейдите в директорию с Django проектом и запустите сервер:

```bash
cd main_site
python manage.py runserver
```

Сайт будет доступен по адресу: http://127.0.0.1:8000

### Запуск парсера

Парсер запускается автоматически через веб-интерфейс при нажатии кнопки "Начать парсинг".

Либо можно запустить вручную из корня проекта:

```bash
python parser/main.py <параметры>
```

## Структура проекта

```
.
├── main_site/          # Django проект
│   ├── static/         # Статические файлы (JS, CSS)
│   ├── templates/      # HTML шаблоны
│   └── page/           # Приложение Django
├── parser/             # Парсер данных
│   ├── main.py         # Точка входа парсера
│   ├── spiders/        # Пауки парсера
│   └── cfg/            # Конфигурация парсера
└── requirements.txt    # Зависимости Python
```

## Примечания

- Убедитесь, что виртуальное окружение активировано перед запуском
- Парсер использует файл `parser/art/tenders.json` для вывода результатов
- Для работы парсера может потребоваться дополнительная настройка в зависимости от источника данных
